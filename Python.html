FAQ Click me!

WHAT IS THIS?

ComputerCraftEdu is a modification (mod) that adds programmable turtle robots to the world of Minecraft. Turtles can do all kinds of useful things such as fly, dig, build and set redstone.

ComputerCraftEdu是一種修改版（mod），在Minecraft的世界中增加了可編程的烏龜機器人。烏龜可以做各種有用的事情，例如飛，挖，建和設置紅石。

WHAT KIND OF A COMPUTER DO I NEED FOR THIS?

If your computer can run Minecraft, you can play ComputerCraftEdu. It works with Windows, OS X and Linux. Check the full requirements here.

如果您的電腦可以運行Minecraft，則可以使用ComputerCraftEdu。它適用於Windows，OS X和Linux。在此處檢查全部要求。

DO I NEED TO KNOW PROGRAMMING TO PLAY?

No. The idea behind ComputerCraftEdu is to give you an easy start to understanding the structures and logic behind programming. You start by controlling your robot with a remote by giving simple one-step commands. You’ll start actual programming by dragging and dropping commands to a grid area and gradually move to placing those blocks by just using the keyboard and finally to write the actual code with just text.

DO I NEED SOME SPECIFIC VERSION OF MINECRAFT?

ComputerCraftEdu is based on Minecraft 1.7. Look for 1.8 updates in the future!

ComputerCraftEdu基於Minecraft 1.7。在將來尋找1.8更新！

WAIT, IS THIS FREE?

Yes, ComputerCraftEdu is completely free. As long as you have either MinecraftEdu or regular Minecraft, you’re good to go!

是的，ComputerCraftEdu模組是完全免費的。只要您擁有MinecraftEdu或常規Minecraft，您就可以執行！
